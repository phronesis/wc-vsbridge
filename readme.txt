This plugin enables the connection of Vue Storefront - The first Ecommerce PWA - to WooCommerce.

== Description ==

Vue Storefront is a standalone PWA storefront for your eCommerce, possible to connect with any eCommerce backend (eg. Magento, Pimcore, Wordpress or Prestashop) through the API.


== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the woocommerce-vsbridge to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
